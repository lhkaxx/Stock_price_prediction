# 股票价格预测 — 项目需求文档

> **课题方向**：时序预测类（LSTM / GRU）  
> **预测标的**：贵州茅台（600519）  
> **预测目标**：单步回归 — 用过去 N 天数据预测第 N+1 天收盘价  
> **数据来源**：akshare（公开金融数据接口）  
> **时间范围**：2020 年 1 月 – 2025 年 12 月  

---

## 一、项目概述

本项目基于人工神经网络与深度学习，构建 LSTM 与 GRU 时序预测模型，对贵州茅台（600519）股票价格进行单步回归预测。通过历史 OHLCV 数据结合多项技术指标，预测下一个交易日的收盘价，并对两种模型进行横向对比分析。

## 三、数据预处理

### 3.1 数据划分

采用**时序划分**策略，避免数据泄露：

| 集合 | 比例 | 用途 |
|------|------|------|
| 训练集 | 70% | 模型训练 |
| 验证集 | 15% | 超参数调优 + 早停判断 |
| 测试集 | 15% | 最终评估 |

### 3.2 标准化

采用 **Z-Score 标准化**（StandardScaler），使每维特征均值为 0、方差为 1：

$$\hat{x} = \frac{x - \mu}{\sigma}$$

- 训练集计算 $\mu$ 和 $\sigma$
- 验证集、测试集复用训练集的均值和标准差

### 3.3 序列构造

| 参数 | 值 | 说明 |
|------|-----|------|
| lookback | 30 | 用过去 30 个交易日的数据 |
| step | 1 | 向前预测 1 天 |

滑动窗口构造 supervised 样本：

```
X: [t-29, t-28, ..., t]        (30 天 × 17 特征)
y: Close_t+1                    (1 个标量)
```

## 四、模型设计

### 4.1 共同架构

| 层 | 配置 | 输出形状 |
|----|------|----------|
| Input | lookback=30, features=17 | (batch, 30, 17) |
| LSTM / GRU | units=64, return_sequences=True | (batch, 30, 64) |
| LSTM / GRU | units=32, return_sequences=False | (batch, 32) |
| Dropout | rate=0.2 | (batch, 32) |
| Dense | units=1 | (batch, 1) |

```
Input(30, 17)
      ↓
LSTM/GRU(64, return_sequences=True)     ← 第一层，保留时序
      ↓
LSTM/GRU(32, return_sequences=False)    ← 第二层，压缩到向量
      ↓
Dropout(0.2)                             ← 防过拟合
      ↓
Dense(1)                                 ← 输出下一天收盘价
```

### 4.2 对比方案

| 模型 | 说明 |
|------|------|
| **LSTM** | 长短期记忆网络，通过遗忘门/输入门/输出门控机制捕捉长期依赖 |
| **GRU** | 门控循环单元，简化版 LSTM（重置门 + 更新门），参数更少 |

两个模型使用**完全相同的层数、单元数、Dropout 率、训练配置**，只替换 RNN 单元类型，确保对比公平。

## 六、评估指标

| 指标 | 全称 | 公式 | 意义 |
|------|------|------|------|
| MSE | Mean Squared Error | $\frac{1}{n}\sum(y_i - \hat{y}_i)^2$ | 均方误差，惩罚大偏差 |
| RMSE | Root Mean Squared Error | $\sqrt{MSE}$ | 量纲与原始值一致 |
| MAE | Mean Absolute Error | $\frac{1}{n}\sum\|y_i - \hat{y}_i\|$ | 平均绝对误差，可解释性强 |
| MAPE | Mean Absolute Percentage Error | $\frac{1}{n}\sum\|\frac{y_i-\hat{y}_i}{y_i}\| \times 100\%$ | 百分比误差 |
| R² | Coefficient of Determination | $1 - \frac{SS_{res}}{SS_{tot}}$ | 拟合优度，越接近 1 越好 |

## 九、开发环境方案

本项目支持**两套环境**并行开发，代码完全互通——本地编写调试后直接复制到 AI Studio 即可运行。

### 9.1 统一框架约束

> **关键原则：两套环境必须使用相同的深度学习框架，否则模型代码不互通。**

| 框架选择 | 本地（ml_env） | AI Studio | 推荐度 |
|----------|:---:|:---:|:---:|
| **PyTorch** | `pip install torch` | 创建项目时选 "PyTorch" 内核 | ⭐⭐⭐ 推荐 |
| PaddlePaddle | `pip install paddlepaddle` | 默认内核，无需配置 | ⭐⭐ |

**推荐 PyTorch**：本地 conda 已有 `ml_env`（含 pandas / numpy / scikit-learn / matplotlib），只需补装 `torch`；且 PyTorch 社区资源丰富，就业市场主流。

### 9.2 方案 A：本地 Conda（开发 + 调试）

| 项目 | 说明 |
|------|------|
| 环境名 | `ml_env`（路径 `D:\ML\ml_env`） |
